Hi,

wir sollten die Methoden wie folgt benennen:

Methode get:
------------
  - Parameter $id
  - Return Daten-Array
  - Liefert einen Datensatz anhand des Primärschlüssels zurück

Methode getList:
----------------
  - Parameter $itemType (eventuell nicht immer)
  - Return Daten-Array
  - Liefert eine Liste der Datensätze

Methode create:
---------------
  - Parameter $data
  - Return neuer Primärschlüssel ($id)
  - Erzeugt einen neuen Datensatz in der Datenbank

Methode remove:
---------------
  - Paramter $id
  - Löscht einen Datensatz in der Datenbank

Methode modify:
---------------
  - Parameter $id, $data
  - Verändert einen Datensatz in der Datenbank

Methode exist:
--------------
  - Paramter $id
  - Return true/false [1/0]
  - Gibt zurück, ob ein Datensatz existiert
